Tem_vivagallery - instructions for installation

The file you downloaded contains your source files and a zip file that you can directly install on your Mambo/Joomla server.

To install this template, extract all files, find the tem_vivagallery.zip file and install it on your server.

Configuring the modules

note: Nothing but the main menu and footer will displayed on the homepage.

1. Go to Modules > Site Modules. Click on the Main Menu module. Select "user1" under Position and make sure the Menu Class Suffix box is empty and that the menu is Horizontal.

2. Create a new custom module to display your custom copyright message and apply it to the bottom position

3. Whatever module is published under Position "user2" will be displayed on the right column.

5. Make sure all of the above modules are published. Also make sure you unpublish any other modules under the positions above.

 

All Set! Your template should look exactly like our demo.
